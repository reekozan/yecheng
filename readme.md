# YÈCHÉNG 夜城 

**Projeto Acadêmico | Desenvolvimento Web Responsivo, UI/UX e Identidade de Marca**

Este projeto foi desenvolvido como um trabalho prático focado em aplicar técnicas avançadas de desenvolvimento web estático, design responsivo e experiência do usuário (UX). Como diferencial, o projeto expandiu o escopo técnico tradicional para criar e implementar o branding completo de uma marca autoral.

A **YÈCHÉNG (夜城)** é uma marca fictícia de moda urbana, futurista e utilitária (techwear), inspirada na estética cyberpunk e na atmosfera das metrópoles globais contemporâneas.

---

## Diferencial do Projeto: Branding e Identidade

Além de cumprir os requisitos técnicos de código, o escopo de design foi expandido para criar uma experiência de marca premium completa:
* **Conceito e Nome:** Criação do nome (YÈCHÉNG 夜城) e posicionamento conceitual bilíngue (Português/Inglês).
* **Posicionamento de Luxo (Cyber-Luxury):** A marca foi estruturada sob o conceito de alta-costura futurista, focando no nicho de luxo cyberpunk.
* **Curadoria e Identidades Únicas:** Desenvolvimento de 7 coleções conceituais exclusivas, incluindo uma linha em Edição Limitada. Cada coleção possui sua própria identidade, narrativa e estética de produto.
* **Complexidade do Catálogo:** Criação e distribuição de 9 cards de produtos exclusivos no catálogo. Para garantir o realismo da experiência do usuário, cada produto possui sua própria página de detalhes individualizada com descrições específicas, em vez de reutilizar uma única página genérica para simulação.

---

## Estrutura do Projeto e Páginas

O projeto foi construído de forma totalmente estática e navegável, sendo composto por 3 páginas interligadas por um menu de navegação comum:

### 1. Home / Catálogo (Listagem de Produtos)
* **Estrutura:** Exibição da coleção de itens utilizando layouts modernos. Cada card de produto possui imagem, título, breve descrição e botão de ação.
* **Desafio Responsivo:** Adaptação da quantidade de colunas conforme o dispositivo (1 coluna em telas mobile, progredindo para layouts de múltiplas colunas em desktops).

### 2. Detalhes do Produto
* **Estrutura:** Página simulada ao selecionar um item do catálogo. Apresenta foto ampliada do produto, título em destaque, descrição detalhada, especificações técnicas e botões de interação ("Adicionar ao carrinho").
* **Desafio Responsivo:** Mudança de disposição estrutural — elementos empilhados verticalmente no mobile (imagem no topo) e dispostos lado a lado (grid/flexbox) no desktop.

### 3. Formulário / Checkout
* **Estrutura:** Página com formulário estruturado utilizando múltiplos tipos de inputs (texto, e-mail, telefone, dropdown e campo de texto longo).
* **Desafio Responsivo:** Campos expandidos para ocupar 100% da largura em telas menores para otimizar a experiência de toque, organizando-se em colunas e layouts mais complexos em telas maiores.

---

## Requisitos Técnicos Implementados

* **Abordagem Mobile-First:** O CSS foi prioritariamente estruturado e pensado para telas menores, utilizando Media Queries para adaptar o layout progressivamente para telas maiores.
* **Semântica HTML:** Uso rigoroso e correto de tags semânticas para garantir acessibilidade e boa estruturação do documento.
* **Navegação Adaptativa:** Menu superior funcional para transição entre as 3 páginas, projetado para se transformar em uma lista vertical ou formato compacto em dispositivos mobile.
* **Tecnologias:** HTML5, CSS3 (Flexbox e Grid) e implementações pontuais de JavaScript para dinamismo da interface.
